Bulk Import

With this Import option, all your existing lists will be unchanged. All imported lists will be added as new lists.
With this Import and merge option, all imported lists will either appear as new lists or be merged with existing lists with the same exact name if a match exists.
If you have multiple lists with the same name, only your first list will be merged with the new lists. List colors of merged lists will also be changed to the color specified in the import, if different.
With the Replace option, all your existing phrase lists will be deleted and replaced with the imported lists.
